## 🔧 技术细节

### 黑屏问题修复
**根本原因：** Vite 默认使用绝对路径 `/assets/...`，在 Electron 中无法加载

**解决方案：**
```typescript
// vite.config.ts
export default defineConfig({
  base: './',  // 使用相对路径
  // ...
});
```

### 关键修改
1. ✅ `vite.config.ts` - 添加 `base: './'`
2. ✅ `electron-main.cjs` - 移除 `openDevTools()`
3. ✅ `electron-main.cjs` - 使用标准标题栏
