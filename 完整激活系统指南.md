## 🔑 激活码生成器使用指南

### 打开生成器

双击打开文件：`activation-code-generator.html`

### 功能说明

#### 1. 单个生成
- 选择"生成单个激活码"
- 可选填写备注信息（如：张三购买，订单号#12345）
- 点击"生成激活码"
- 复制生成的激活码发送给客户

#### 2. 批量生成
- 选择"批量生成激活码"
- 输入生成数量（1-100）
- 可选填写备注信息
- 点击"批量生成"
- 可以：
  - 逐个复制激活码
  - 复制所有激活码
  - 下载为文本文件

### 激活码格式

```
VOICE-XXXXX-XXXXX-XXXXX-XXXXX
```

示例：
```
VOICE-K3N7P-9H2TL-R6XMW-8QD4Y
```

### 分发流程

1. **客户购买** → 使用生成器创建激活码
2. **记录信息** → 在备注中填写客户信息和订单号
3. **发送激活码** → 通过邮件/微信等方式发送给客户
4. **保存记录** → 使用"下载激活码列表"保存记录

## 💻 用户使用流程

### 试用模式（未激活）

1. **首次打开软件**
   - 顶部显示"试用模式"横幅
   - 提示："还可以处理 2 个文件"

2. **处理第1个文件**
   - 上传音频文件
   - 点击"开始处理"
   - 处理完成后，剩余次数变为 1

3. **处理第2个文件**
   - 上传新文件
   - 点击"开始处理"
   - 处理完成后，剩余次数变为 0
   - 2秒后自动弹出激活对话框

4. **尝试第3次处理**
   - 上传文件
   - 点击"开始处理"
   - 立即被拦截，弹出激活对话框
   - 提示："试用次数已用完，请购买激活码"

### 激活流程

1. **打开激活对话框**
   - 试用次数用完时自动弹出
   - 或点击左下角【激活】按钮

2. **输入激活码**
   - 在输入框中输入您提供的激活码
   - 格式：VOICE-XXXXX-XXXXX-XXXXX-XXXXX

3. **点击激活**
   - 点击"立即激活"按钮
   - 等待1秒（模拟网络验证）

4. **激活结果**
   - 成功：显示绿色提示"激活成功！享受完整功能吧"
   - 失败：显示红色错误提示

5. **自动关闭**
   - 激活成功后，2秒后自动关闭对话框
   - 试用横幅消失
   - 可以无限次使用所有功能

### 激活后

- ✅ 无限制处理音频文件
- ✅ 所有功能完全解锁
- ✅ 刷新页面后激活状态保持
- ✅ 不显示试用横幅

## 🔧 技术说明

### 激活码验证逻辑

当前版本使用**本地验证**：

```typescript
// 验证规则
1. 格式检查：VOICE-XXXXX-XXXXX-XXXXX-XXXXX
2. 长度检查：每段必须是5个字符
3. 字符检查：只允许大写字母和数字
4. 自动通过：所有格式正确的激活码都能通过
```

### 升级为在线验证（推荐）

如果您想要更安全的验证，可以：

1. **搭建后端API**
   ```javascript
   // 后端接口示例
   POST https://your-api.com/activate
   {
     "activationCode": "VOICE-...",
     "deviceId": "...",
     "version": "1.0.0"
   }
   ```

2. **修改验证代码**
   - 文件：`src/lib/licenseManager.ts`
   - 方法：`verifyActivationCodeOnline`
   - 取消注释API调用示例代码

3. **激活码数据库**
   - 记录生成的激活码
   - 记录激活状态
   - 限制激活次数
   - 支持激活码撤销

### 数据存储位置

激活信息存储在浏览器的 localStorage：

```javascript
// 激活状态
localStorage.getItem('miaoyin_license')

// 试用次数
localStorage.getItem('miaoyin_trial_count')
```

清除激活（测试用）：
```javascript
localStorage.removeItem('miaoyin_license');
localStorage.removeItem('miaoyin_trial_count');
location.reload();
```
