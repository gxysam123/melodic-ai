## 📋 安装和使用

### macOS 安装

#### 主应用
```bash
# 1. 双击安装 DMG
open miaoyin-ai-audio-center/release/声音处理工具-1.0.0-arm64.dmg

# 2. 拖拽到 Applications

# 3. 解除限制（首次必须）

# 4. 启动
open /Applications/声音处理工具.app
```

## 🎯 功能使用指南

### 1. 试用模式
- 新安装的应用提供 **2次免费试用**
- 每次处理成功会消耗 1 次
- 点击"停止处理"不消耗次数
- 试用次数用完后需要激活码

### 2. 获取激活码
使用激活码生成器：

1. 打开"激活码生成器"
2. 选择激活类型：
   - 永久激活
   - 临时激活（设置天数）
3. 设置使用次数（可选）
4. 添加备注信息
5. 点击"生成激活码"
6. 复制生成的激活码

### 3. 激活应用
1. 打开"声音处理工具"
2. 试用次数用完后会弹出激活对话框
3. 或点击左下角 "i" → "激活"
4. 输入激活码
5. 点击"激活"按钮

### 4. 音频处理

#### 智能降噪
1. 上传音频文件（支持 MP3, WAV, FLAC, OGG）
2. 选择"智能降噪"
3. 点击"开始处理"
4. 等待处理完成
5. 点击"下载"获取处理后的文件

#### 多音轨分离
1. 上传音频文件
2. 选择"多音轨分离"
3. 选择分离模式：
   - 4轨：人声、鼓、贝斯、其他
   - 2轨：人声、伴奏
4. 点击"开始处理"
5. 下载分离后的音轨

### 5. 开发者选项
- 点击左下角 "i" 图标
- 滚动到底部
- 点击"开发者：重置激活状态"
- 用于测试或重置试用次数

## 🔧 常见问题

### Q1: macOS 提示"应用已损坏"？
```bash
sudo xattr -rd com.apple.quarantine /Applications/声音处理工具.app
```

### Q2: Windows 提示"Windows 已保护你的电脑"？
点击"更多信息" → "仍要运行"

### Q3: 找不到 Python？
**macOS:**
```bash
brew install python@3.12
```

**Windows:**
下载安装：https://www.python.org/downloads/

### Q4: 首次处理很慢？
首次处理会下载 AI 模型（~600MB），后续会快很多。

### Q5: 激活码无效？
- 检查是否复制完整
- 检查是否过期
- 检查使用次数是否用完

### Q6: 如何卸载？
**macOS:**
```bash
rm -rf /Applications/声音处理工具.app
rm -rf ~/Library/Application\ Support/声音处理工具
```

**Windows:**
删除解压的文件夹即可

## 🔐 安全说明

### 激活系统
- 激活码存储在本地
- 不会上传到服务器
- 可以离线验证

### 隐私保护
- 所有音频处理都在本地完成
- 不会上传音频文件
- 不会收集用户数据

### 文件存储
- 上传的文件临时存储在：
  - macOS: `/var/folders/...`
  - Windows: `%TEMP%`
- 处理完成后自动删除
