## 测试场景

### 场景1：首次使用（试用模式）
```
1. 打开应用
   ✅ 显示"试用模式"横幅
   ✅ 显示"剩余 2 次使用机会"

2. 处理第1个文件
   ✅ 处理成功
   ✅ 剩余次数：1

3. 处理第2个文件
   ✅ 处理成功
   ✅ 剩余次数：0
   ✅ 2秒后弹出激活对话框

4. 尝试第3次处理
   ❌ 立即被拦截
   ✅ 弹出激活对话框
```

### 场景2：激活软件
```
1. 点击【激活】按钮
   ✅ 弹出激活对话框

   ❌ 显示格式错误提示

3. 输入测试激活码
   ✅ 激活成功
   ✅ 2秒后自动关闭对话框
   ✅ 试用横幅消失

4. 刷新页面
   ✅ 激活状态保持
```
