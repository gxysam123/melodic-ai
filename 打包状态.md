## 💡 关于应用图标

当前使用 Electron 默认图标。如需自定义图标：

### 1. 准备图标文件
- 创建 1024x1024px 的 PNG 图片
- 保存为 `assets/icon.png`

### 2. 生成 ICNS 文件
```bash
# 使用在线工具转换
# https://cloudconvert.com/png-to-icns

# 或使用 macOS 自带工具
mkdir icon.iconset
sips -z 512 512 assets/icon.png --out icon.iconset/icon_512x512.png
sips -z 1024 1024 assets/icon.png --out icon.iconset/icon_512x512@2x.png
iconutil -c icns icon.iconset -o assets/icon.icns
rm -rf icon.iconset
```

### 3. 重新打包
```bash
npm run electron:build-dmg
```
