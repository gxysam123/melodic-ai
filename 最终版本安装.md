## 🎨 自定义图标（可选）

当前使用默认 Electron 图标。如需自定义：

### 1. 准备图标文件
```bash
# 创建 1024x1024px 的 PNG 图片
# 保存为 assets/icon.png
```

### 2. 生成 ICNS
```bash
cd ~/Desktop/L/11-AI-project/09-melodic-ai-new/miaoyin-ai-audio-center

# 在线转换工具
# https://cloudconvert.com/png-to-icns
# 下载后保存为 assets/icon.icns
```

### 3. 更新配置
编辑 `package.json`，取消注释图标配置：
```json
{
  "build": {
    "mac": {
      "icon": "assets/icon.icns"
    }
  }
}
```

### 4. 重新打包
```bash
npm run electron:build-dmg
```
