# 妙音AI 激活系统说明

## 主要功能

### 1. 试用限制
- ✅ **未激活时可以试用处理 2 次**
- 每次点击"开始处理"并成功处理音频文件，会消耗 1 次试用机会
- 试用次数用完后，必须激活才能继续使用

### 2. 激活功能
- ✅ **点击左下角【激活】按钮，弹出激活码输入对话框**
- 输入格式：`MIAOYIN-XXXXX-XXXXX-XXXXX-XXXXX`
- 激活成功后，可以永久无限制使用所有功能

### 3. 试用提示
- 在未激活状态下，顶部会显示试用横幅
- 横幅显示剩余试用次数
- 提供"立即激活"快捷按钮

### 4. 处理拦截
- 试用次数用完后，点击"开始处理"会被拦截
- 自动弹出激活对话框，提示用户激活

## 测试激活码

为了方便测试，系统内置了两个测试激活码：

1. `MIAOYIN-DEMO1-DEMO2-DEMO3-DEMO4`
2. `MIAOYIN-TEST1-TEST2-TEST3-TEST4`

## 使用流程

### 首次使用（试用模式）
1. 打开软件，看到"试用模式"横幅，显示"还可以处理 2 个文件"
2. 上传音频文件
3. 点击"开始处理"
4. 处理完成后，剩余次数变为 1
5. 再次处理一个文件，剩余次数变为 0
6. 第三次尝试处理时，会弹出激活对话框

### 激活软件
1. 点击左下角【激活】按钮
2. 在弹出的对话框中输入激活码
3. 点击"立即激活"
4. 激活成功后，试用横幅消失
5. 可以无限次处理音频文件

## 数据存储

系统使用 localStorage 存储激活信息：

- `miaoyin_license`: 存储激活状态和激活码
- `miaoyin_trial_count`: 存储已使用的试用次数

## 安全特性

1. **设备绑定**: 激活码与设备ID绑定，防止多设备使用
2. **格式验证**: 严格的激活码格式检查
3. **本地存储**: 激活信息安全存储在本地
4. **试用限制**: 严格的试用次数控制

## 开发提示

在实际部署时，需要：

1. 实现真实的后端激活API
2. 连接到激活服务器验证激活码
3. 实现激活码生成和管理系统
4. 添加购买激活码的支付流程
5. 移除测试激活码

当前版本使用本地验证，仅供演示和测试。
