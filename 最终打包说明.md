## ⚠️ 注意事项

### macOS
1. 首次运行需要解除限制：
   ```bash
   sudo xattr -rd com.apple.quarantine /Applications/声音处理工具.app
   ```

2. 应用未签名，会有安全提示

### Windows
1. 需要安装 Python 3.x
2. 首次运行可能有 Windows Defender 警告
3. 便携版无需安装，解压即用

### 通用
1. 首次处理会下载 AI 模型（~600MB）
2. 需要足够的磁盘空间
3. 建议至少 8GB 内存
