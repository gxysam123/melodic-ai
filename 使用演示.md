## 脚本说明

### start.sh - 一键启动
```bash
./start.sh
```
**功能**：
- ✅ 自动检查并安装所有依赖
- ✅ 同时启动前后端服务
- ✅ 实时监控服务状态
- ✅ 显示访问地址和日志路径
- ✅ 按 Ctrl+C 优雅停止

### stop.sh - 停止服务
```bash
./stop.sh
```
**功能**：
- 🛑 停止后端服务
- 🛑 停止前端服务
- 🧹 清理残留进程
- 🗑️  删除 PID 文件

### logs.sh - 查看日志
```bash
./logs.sh
```
**功能**：
- 📋 选择查看后端或前端日志
- 📊 实时跟踪日志输出
- 🔍 快速定位问题

## 常用命令

```bash
# 启动
./start.sh

# 停止
./stop.sh

# 查看日志
./logs.sh

# 手动查看后端日志
tail -f miaoyin-ai-audio-center/backend.log

# 手动查看前端日志
tail -f miaoyin-ai-audio-center/frontend.log

# 检查服务状态
curl http://localhost:5001/api/health
curl http://localhost:3000

# 查看进程
ps aux | grep "backend_server\|vite"

# 杀死进程（如果脚本失败）
pkill -f "python3 backend_server.py"
pkill -f "vite.*3000"
```

## 快捷键

在运行 `./start.sh` 时：
- `Ctrl+C` - 停止所有服务

在运行 `./logs.sh` 时：
- `Ctrl+C` - 退出日志查看

## 提示

1. **首次启动可能较慢**，需要下载 Demucs 模型（约 300MB）
2. **使用 GPU** 可以显著提升处理速度（需要 NVIDIA 显卡）
3. **日志文件** 会持续增长，定期清理可节省空间
4. **网络访问** 功能让你可以在同一局域网的其他设备上使用
