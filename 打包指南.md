## 📦 开始打包

```bash
cd ~/Desktop/L/11-AI-project/09-melodic-ai-new/miaoyin-ai-audio-center

# 运行打包脚本
./electron-build.sh
```

### 选择打包类型

脚本会提示你选择：
1. **DMG 安装包**（推荐）- 双击即可安装
2. **ZIP 压缩包** - 解压后直接使用
3. **两者都打包** - 同时生成 DMG 和 ZIP

**推荐选择 1**

## 🎯 安装应用

### 方式 1：从 DMG 安装（推荐）

1. **双击** `妙音AI-1.0.0.dmg` 打开
2. **拖拽** 应用图标到 `Applications` 文件夹
3. **打开** Finder → 应用程序 → 妙音AI
4. **首次运行**可能需要在「系统偏好设置 → 安全性与隐私」中允许

### 方式 2：从 ZIP 安装

1. **解压** `妙音AI-1.0.0-mac.zip`
2. **拖拽** 解压出的 `妙音AI.app` 到 `Applications` 文件夹
3. **运行** 应用

## 🔐 macOS 安全提示

如果遇到"无法打开应用"的警告：

### 方法 1：系统设置（推荐）
1. 打开「系统偏好设置」
2. 进入「安全性与隐私」
3. 点击「仍要打开」

### 方法 2：命令行移除隔离属性
```bash
xattr -cr /Applications/妙音AI.app
```

### 方法 3：右键打开
1. 在应用程序中找到「妙音AI」
2. **右键点击** → 打开
3. 在弹出对话框中点击「打开」

## ⚙️ 手动打包命令

如果脚本失败，可以手动运行：

```bash
# 清理
rm -rf dist release

# 构建前端
npm run build

# 打包 DMG（仅限当前架构）
electron-builder --mac dmg

# 打包所有格式
electron-builder --mac

# 打包 ZIP
electron-builder --mac zip

# 打包 DMG + ZIP（推荐）
electron-builder --mac dmg zip
```

## 📊 架构说明

打包会生成多个文件：

### Universal（通用版）
- `妙音AI-1.0.0.dmg` - 同时支持 Intel 和 Apple Silicon
- 文件较大，但兼容性最好

### 分架构版本
- `妙音AI-1.0.0-arm64.dmg` - Apple Silicon (M1/M2/M3) 专用
- `妙音AI-1.0.0-x64.dmg` - Intel 芯片专用

**推荐分发 Universal 版本**，除非文件太大。

## 📏 文件大小说明

预计大小：
- **DMG 安装包**：约 100-300 MB
- **ZIP 压缩包**：约 80-250 MB

文件大小取决于：
- Node.js 依赖数量
- Python 包（Demucs 模型会包含在内）
- 资源文件

## 💡 优化建议

### 减小包体积
1. 移除不必要的依赖
2. 使用 `files` 配置过滤文件
3. 不打包开发依赖

### 加快打包速度
1. 使用 `--dir` 先测试打包
2. 缓存 node_modules
3. 使用本地构建而非 CI
