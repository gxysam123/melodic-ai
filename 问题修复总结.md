### 4. ✅ 处理按钮bug修复
**问题：** 不管激活还是未激活，都不能点击"开始处理"

**原因：**
- 按钮disabled条件：`!taskId`
- 演示文件不生成taskId
- 导致按钮永远被禁用

**解决方案：**
```typescript
// 修改前
if (uploadFile) {
  setTaskId(response.task_id);
} else {
  // 演示文件不设置taskId
}

// 修改后
if (uploadFile) {
  setTaskId(response.task_id);
} else {
  // 演示文件也生成taskId
  const demoTaskId = 'demo-' + Date.now();
  setTaskId(demoTaskId);
}
```
