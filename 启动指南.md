# 妙音AI - 快速启动指南

## 🚀 一键启动

在项目根目录运行：

```bash
./start.sh
```

脚本会自动：
- ✅ 检查环境（Python、Node.js、npm）
- ✅ 检查并安装依赖
- ✅ 启动后端服务（端口 5001）
- ✅ 启动前端服务（端口 3000）
- ✅ 显示访问地址和日志路径

## 🛑 停止服务

```bash
./stop.sh
```

或者在运行 `start.sh` 的终端按 `Ctrl+C`

## 📋 查看日志

```bash
./logs.sh
```

选择要查看的日志：
1. 后端日志
2. 前端日志
3. 同时查看（分屏）

## 📁 项目结构

```
09-melodic-ai-new/
├── start.sh                    # 一键启动脚本

    ├── backend.log             # 后端日志（运行时生成）
    ├── frontend.log            # 前端日志（运行时生成）

```

## 🌐 访问地址

启动成功后，访问：

- **本地**: http://localhost:3000
- **网络**: http://你的IP:3000（局域网内其他设备可访问）
- **API**: http://localhost:5001/api

## 🔧 手动启动（如果脚本失败）

### 1. 启动后端

```bash
cd miaoyin-ai-audio-center
python3 backend_server.py
```

### 2. 启动前端（新开终端）

```bash
cd miaoyin-ai-audio-center
npm run dev
```

## 💡 功能说明

### 1. 智能降噪
- 清理音频中的背景噪音
- 保留清晰人声
- 适合录音、会议、播客

### 2. 多音轨分离
- 分离人声、鼓声、贝斯、其他伴奏
- 支持混音重组
- 适合音乐制作、卡拉OK

### 3. 人声分离（基础）
- 提取纯净人声
- 分离背景音乐
- 适合配音、翻唱

## 🐛 常见问题

### 1. 端口被占用

**错误**: `Address already in use`

**解决**:
```bash
# 查找占用端口的进程
lsof -i :5001  # 后端端口
lsof -i :3000  # 前端端口

# 杀死进程
kill -9 <PID>
```

### 2. Python 依赖安装失败

**解决**:
```bash
# 升级 pip
pip3 install --upgrade pip

# 重新安装
pip3 install -r requirements.txt
```

### 3. 前端无法连接后端

**检查**:
- 后端是否正常运行（访问 http://localhost:5001/api/health）
- 查看后端日志：`tail -f miaoyin-ai-audio-center/backend.log`

### 4. 处理速度慢

**优化**:
- 使用 GPU 加速（需要 NVIDIA 显卡和 CUDA）
- 减小音频文件大小
- 关闭其他占用资源的程序

## 📊 性能说明

### CPU 模式
- 人声分离: 约 1-3x 实时（10分钟音频 → 10-30分钟处理）
- 多音轨分离: 约 2-5x 实时（10分钟音频 → 20-50分钟处理）

### GPU 模式（NVIDIA）
- 人声分离: 约 0.3-1x 实时（10分钟音频 → 3-10分钟处理）
- 多音轨分离: 约 0.5-2x 实时（10分钟音频 → 5-20分钟处理）
