## 🚀 使用流程

### 用户侧

1. **安装应用**
   - macOS: 双击 DMG，拖拽到 Applications
   - Windows: 解压 ZIP，运行 .exe

2. **首次运行**
   - macOS: 需要运行 `sudo xattr -rd com.apple.quarantine /Applications/声音处理工具.app`
   - Windows: 可能需要"仍要运行"

3. **试用**
   - 免费试用 2 次
   - 每次成功处理消耗 1 次

4. **激活**
   - 试用结束后输入激活码
   - 或点击"激活"按钮

### 管理员侧

1. **打开激活码生成器**
   - macOS: 运行 DMG 安装的应用
   - 或浏览器打开 HTML 文件

2. **生成激活码**
   - 选择激活类型（永久/临时）
   - 设置使用次数（可选）
   - 点击"生成激活码"

3. **分发激活码**
   - 复制给用户
   - 或批量生成导出

## 📝 注意事项

### macOS
1. **必须解除限制：**
   ```bash
   sudo xattr -rd com.apple.quarantine /Applications/声音处理工具.app
   ```

2. **应用未签名**
   - 会有"无法验证开发者"提示
   - 正常现象，不影响使用

3. **架构：**
   - 仅支持 Apple Silicon (ARM64)
   - Intel Mac 可能无法运行

### Windows
1. **Python 依赖**
   - 需要安装 Python 3.x
   - 应用会自动检测

2. **便携版说明**
   - 无需安装，解压即用
   - 所有文件在 win-unpacked 文件夹中
   - 不要删除任何 DLL 文件

3. **Windows Defender**
   - 可能会提示不信任
   - 选择"仍要运行"

### 通用
1. **首次运行**
   - 会下载 AI 模型（~600MB）
   - 需要网络连接
   - 需要足够的磁盘空间

2. **系统要求**
   - 最低 8GB 内存
   - 5GB 可用磁盘空间
   - i5 或 M1 以上 CPU

## 🔐 激活码使用说明

### 激活码格式
```
XXXX-XXXX-XXXX-XXXX-XXXX-XXXX
```
- 30 个字符
- 数字 + 大写字母
- 用短横线分隔

### 激活类型

#### 1. 永久激活
- 不设置过期时间
- 可选使用次数限制
- 推荐给长期用户

#### 2. 临时激活
- 设置过期天数（如 30 天）
- 可选使用次数限制
- 推荐给试用用户

#### 3. 限次激活
- 不限时间
- 设置使用次数（如 100 次）
- 推荐给批量用户
