# 激活功能测试清单

## 测试前准备
1. 打开浏览器开发者工具（F12）
2. 进入 Application > Local Storage
3. 清除 `miaoyin_license` 和 `miaoyin_trial_count`
4. 刷新页面

## 测试步骤

### ✅ 测试1：验证试用次数为 2 次
1. 打开应用，应该看到"试用模式"横幅
2. 横幅显示："还可以处理 **2** 个文件"
3. 日志显示："⚠️ 试用模式：剩余 2 次使用机会"

### ✅ 测试2：第一次处理
1. 上传一个音频文件
2. 点击"开始处理"
3. 等待处理完成
4. 查看横幅，应该显示："还可以处理 **1** 个文件"

### ✅ 测试3：第二次处理
1. 删除之前的文件，上传新文件
2. 点击"开始处理"
3. 等待处理完成
4. 查看横幅，应该显示："还可以处理 **0** 个文件"
5. 2秒后应该自动弹出激活对话框

### ✅ 测试4：试用次数用完拦截
1. 删除之前的文件，上传新文件
2. 点击"开始处理"
3. 应该立即被拦截，弹出激活对话框
4. 日志显示："❌ 试用次数已用完，请购买激活码"

### ✅ 测试5：激活按钮
1. 点击左下角的【激活】按钮（紫色渐变）
2. 应该弹出激活对话框
3. 对话框标题："激活妙音AI"
4. 显示激活码输入框

### ✅ 测试6：激活码格式验证
1. 在激活对话框输入错误格式的激活码：`123456`
2. 点击"立即激活"
3. 应该显示错误提示："激活码格式错误，正确格式: MIAOYIN-XXXXX-XXXXX-XXXXX-XXXXX"

### ✅ 测试7：使用测试激活码
1. 在激活对话框输入：`MIAOYIN-DEMO1-DEMO2-DEMO3-DEMO4`
2. 点击"立即激活"
3. 应该显示成功提示："激活成功！享受完整功能吧"
4. 2秒后自动关闭对话框
5. 试用横幅消失
6. 日志显示："🎉 软件激活成功！现在可以无限制使用所有功能"

### ✅ 测试8：激活后无限使用
1. 激活成功后
2. 上传并处理多个文件
3. 不应该有任何试用限制
4. 不应该显示试用横幅

### ✅ 测试9：激活信息持久化
1. 激活成功后
2. 刷新页面
3. 激活状态应该保持
4. 不显示试用横幅
5. 日志显示："✅ 软件已激活，享受完整功能"

### ✅ 测试10：关于按钮
1. 点击左下角的【i】图标（关于按钮）
2. 应该弹出关于对话框
3. 显示软件版本和信息

## 重置测试环境

如果需要重新测试，执行以下步骤：

1. 打开浏览器开发者工具（F12）
2. 进入 Console 标签
3. 输入并执行：
```javascript
localStorage.removeItem('miaoyin_license');
localStorage.removeItem('miaoyin_trial_count');
location.reload();
```

或者在 Application > Local Storage 中手动删除这两个键。
